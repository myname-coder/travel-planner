🔥 เชื่อมต่อ Firebase เพื่อแชร์เรียลไทม์
คุณสามารถแชร์ทริปให้เพื่อนช่วยกันแต่งทริปได้แบบ Real-time! เพียงเปิดใช้งาน Cloud Firestore (ฟรี) แล้วนำ Config มาใส่ในโค้ด:
  1. สมัคร/ล็อกอินที่ Firebase Console
  2. สร้างโครงการใหม่ (Create a project)
  3. สร้างฐานข้อมูล Cloud Firestore (เลือก Start in test mode)
  4. สร้างแอพพลิเคชันประเภท Web (</>) ในโครงการ
  5. คัดลอก firebaseConfig ที่ได้ ไปแก้ไขในไฟล์ index.html
const firebaseConfig = {
  apiKey: "AIzaSy...",
  authDomain: "...",
  projectId: "...",
  ...
};
